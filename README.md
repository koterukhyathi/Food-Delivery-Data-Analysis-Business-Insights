# Food-Delivery-Data-Analysis-Business-Insights
This project performs exploratory data analysis (EDA) on food delivery and restaurant order datasets to uncover customer behavior, operational trends, and business insights using Python.


The primary objective was to transform raw operational data into actionable insights using Python data analysis libraries and visualization tools.

⸻

🎯 Objectives

* Perform data cleaning and preprocessing
* Conduct exploratory data analysis (EDA)
* Analyze customer ordering patterns
* Identify trends in ratings, pricing, and delivery times
* Visualize insights for business decision-making

⸻

🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

⸻

📊 Key Analysis Performed

Data Cleaning & Preprocessing

* Handling missing values
* Removing inconsistencies
* Formatting and transforming data

Exploratory Data Analysis

* Order distribution analysis
* Delivery time analysis
* Cuisine popularity trends
* Restaurant rating analysis
* Cost and pricing distribution

Data Visualization

* Histograms
* Box plots
* Count plots
* Correlation analysis
* Trend visualizations

⸻

📈 Key Insights

* Weekend orders showed increased customer activity
* Certain cuisines consistently received higher ratings
* Peak delivery times impacted delivery efficiency
* Pricing and discounts influenced ordering behavior
* Customer satisfaction patterns varied across restaurants
